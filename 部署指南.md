# 红运网站部署指南（腾讯云 + 宝塔面板）

## 📋 前提条件

- 腾讯云服务器已安装宝塔面板
- 本地已安装 Node.js（建议 18.x 或更高版本）
- 有服务器 SSH 登录权限或宝塔面板登录权限

---

## 第一步：本地构建项目

### 1. 打开终端，进入项目目录

```bash
cd /Users/qiaoya/Downloads/dev-hongyun-2026-03-10/hongyun-react
```

### 2. 安装依赖（如果还没安装）

```bash
npm install
```

### 3. 构建生产版本

```bash
npm run build
```

**执行成功后**，会在项目根目录生成 `dist` 文件夹，这就是要上传到服务器的文件。

**检查 dist 文件夹内容：**
```
dist/
├── index.html          # 主页面
├── assets/             # 所有 CSS、JS、图片资源
│   ├── index-xxx.js
│   ├── index-xxx.css
│   └── img/
└── ...
```

---

## 第二步：登录宝塔面板

### 1. 打开浏览器，访问宝塔面板

通常地址是：`http://你的服务器IP:8888`

例如：`http://123.456.789.0:8888`

### 2. 输入宝塔账号密码登录

如果忘记密码，可以通过 SSH 连接服务器后执行：
```bash
bt default
```
查看默认账号密码。

---

## 第三步：在宝塔面板创建网站

### 1. 点击左侧菜单「网站」

### 2. 点击「添加站点」按钮

### 3. 填写网站信息

- **域名**：填写你的域名，例如 `www.hongyun.com`（如果暂时没有域名，可以先填服务器 IP）
- **备注**：红运官网
- **根目录**：保持默认或自定义，例如 `/www/wwwroot/hongyun`
- **FTP**：不需要勾选
- **数据库**：不需要勾选（纯静态网站）
- **PHP 版本**：选择「纯静态」

### 4. 点击「提交」创建网站

---

## 第四步：上传网站文件

### 方法一：通过宝塔面板上传（推荐新手）

1. 在「网站」列表中，找到刚创建的网站
2. 点击「根目录」列的路径（例如 `/www/wwwroot/hongyun`）
3. 进入文件管理器
4. **删除默认文件**：
   - 选中所有文件（如 `index.html`、`.user.ini` 等）
   - 点击「删除」
5. **上传 dist 文件夹内容**：
   - 点击「上传」按钮
   - 选择本地 `dist` 文件夹内的**所有文件和文件夹**
   - 等待上传完成（可能需要几分钟）

**重要**：上传的是 `dist` 文件夹**里面**的内容，不是 `dist` 文件夹本身！

上传后的目录结构应该是：
```
/www/wwwroot/hongyun/
├── index.html
├── assets/
│   ├── index-xxx.js
│   ├── index-xxx.css
│   └── img/
└── ...
```

### 方法二：通过 FTP 工具上传（可选）

如果文件较多，可以使用 FileZilla 等 FTP 工具：

1. 在宝塔面板「FTP」菜单中创建 FTP 账号
2. 使用 FTP 工具连接服务器
3. 上传 `dist` 文件夹内容到网站根目录

---

## 第五步：配置网站（重要）

### 1. 设置伪静态规则（支持 React Router）

1. 在「网站」列表中，点击网站的「设置」按钮
2. 点击「伪静态」标签
3. 在输入框中粘贴以下规则：

```nginx
location / {
  try_files $uri $uri/ /index.html;
}
```

4. 点击「保存」

**这一步非常重要**，否则刷新页面或直接访问子路由会出现 404 错误。

### 2. 配置 HTTPS（可选但推荐）

如果你有域名并且想使用 HTTPS：

1. 点击「SSL」标签
2. 选择「Let's Encrypt」（免费证书）
3. 输入邮箱
4. 点击「申请」
5. 申请成功后，开启「强制 HTTPS」

---

## 第六步：访问网站

### 1. 打开浏览器访问

- 如果使用域名：`http://www.hongyun.com`
- 如果使用 IP：`http://你的服务器IP`

### 2. 检查网站功能

- 首页是否正常显示
- 点击导航菜单，检查各个页面是否正常
- 刷新页面，确认不会出现 404 错误
- 检查图片是否正常加载

## 🔧 常见问题

### 1. 访问网站显示 403 或 404

**原因**：文件权限问题或文件未正确上传

**解决**：
- 在宝塔文件管理器中，右键点击网站根目录
- 选择「权限」→ 设置为 `755`
- 勾选「应用到子目录」
- 确认 `index.html` 文件存在于根目录

### 2. 刷新页面出现 404

**原因**：未配置伪静态规则

**解决**：按照「第五步 - 设置伪静态规则」操作

### 3. 图片无法显示

**原因**：图片路径问题或未上传

**解决**：
- 确认 `assets/img/` 文件夹已上传
- 检查浏览器控制台（F12）查看具体错误
- 确认图片文件名大小写一致

### 4. CSS 样式丢失

**原因**：资源路径问题

**解决**：
- 确认 `assets/` 文件夹完整上传
- 检查 `vite.config.js` 中的 `base` 配置（应该是 `/` 或留空）

### 5. 域名无法访问但 IP 可以

**原因**：域名未解析或未备案

**解决**：
- 在域名服务商处添加 A 记录，指向服务器 IP
- 如果是国内服务器，确保域名已备案

## 🔄 后续更新网站

当你修改了代码需要重新部署时：

1. 本地执行 `npm run build` 重新构建
2. 登录宝塔面板
3. 进入网站根目录
4. **删除旧文件**（保留 `.user.ini` 和 `.htaccess` 如果有）
5. 上传新的 `dist` 文件夹内容
6. 清除浏览器缓存，刷新查看

---

## 📞 需要帮助？

如果遇到问题：

1. 检查宝塔面板「日志」→「网站日志」查看错误信息
2. 浏览器按 F12 打开开发者工具，查看 Console 和 Network 标签
3. 确认服务器防火墙已开放 80 和 443 端口
